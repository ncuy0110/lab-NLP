
# Crawl 1000 posts from Viblo.asia



## Build and run

To deploy this project run

```bash
  python -m pip install virtualenv
```

```bash
  python -m venv venv
```
```
  Windows
  source venv/Scripts/activate
  or Unix
  source venv/bin/active
```

```
  pip install -r requirements.txt
```

```
  python main.py
```
